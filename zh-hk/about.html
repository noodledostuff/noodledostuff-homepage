Profile

把創作系統真正砌到可用。

我喜歡推動邊界：由零開始建立 LED stage、用有限預算為 VTuber 製作 virtual concert，並把 real-time rendering、generative AI 和 storytelling 接成可以承受現場壓力的體驗。

不在測試硬件或寫 code 的時候，我多半都在試新 workflow，或者協助電影、教育、互動技術、活動製作及軟件團隊把想法變成能運作的系統。
